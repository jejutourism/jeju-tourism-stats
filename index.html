Supabase 클라우드 연결 설정
아래 2단계를 완료하면 모든 직원이 설정 없이 자동으로 데이터를 볼 수 있습니다
● 미연결
① Supabase URL / Key 입력 후 연결 테스트
② 배포용 파일 다운로드 → Netlify 재배포 (직원 공유 적용)
연결 테스트 성공 후 이 버튼을 누르면 Supabase 키가 포함된 index.html이 다운로드됩니다.
이 파일을 Netlify에 재배포하면 직원들이 접속만 해도 자동으로 데이터가 표시됩니다.
PC가 바뀌어도 재설정 불필요 — 직원은 guest 로그인만 하면 됩니다.
